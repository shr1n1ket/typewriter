# Typewriter

A minimalist, browser-based typewriter inspired by mechanical writing machines.

This project simulates the feel of a real typewriter with moving paper, a blinking cursor, and authentic typing sounds.  
Built as a personal learning project using plain HTML, CSS, and JavaScript.

## Features
- Realistic paper movement while typing
- Mechanical typewriter sound effects
- Minimalist UI inspired by classic typewriters
- Works entirely in the browser (no frameworks)
